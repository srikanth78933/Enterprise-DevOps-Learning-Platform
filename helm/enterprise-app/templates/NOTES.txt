If Argo CD manages this release (Project 4+), don't `helm upgrade`
directly - it fights the continuous reconciliation loop and gets reverted
by `selfHeal: true`. Change the image tag by editing
helm/enterprise-app/values-images/backend.yaml or frontend.yaml, commit,
and push - Argo CD picks it up automatically (or Jenkins does this for
you, see backend/Jenkinsfile's "Update GitOps Values" stage).

See helm/enterprise-app/README.md for secrets setup (required before first install).
